# Twist: customizable christmas decoration

Twist customizable christmas decoration

## customizable variables

- `NbArms` = 5; // [2:1:20]
- `Height` = 200; // [1:1:200]
- `DistFromBorder` = 30; // [1:1:200]
- `ArmRadius` = 10; // [2:1:20]
- `TopCone_percentHeight` = 85; // [0:5:100]
- `InsideCone` = false;
- `Bottom` = true;
- `TopHook` = true;
