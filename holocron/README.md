# Jedi holocron box multi material

Jedi holocron box.

## Instructions

- print 4 face_side.stl
- print 2 face.stl
- print box_bottom.stl
- print box_top.stl
- glue box_top to one face
- glue box_bottom to one face and 4 face_side
- remove (cut) the red little parts
